per configurare il sito di lineshare, prima bisogna creare una cartella "lineshare" dentro 
la cartella htdocs di xampp.
Successivamente bisogna andare su 127.0.0.1/phpmyadmin e creare un database chiamandolo "lineshare".
Selezionare il database ed importare "lineshare.sql"
digitare sul browser 127.0.0.1/lineshare
